My Projects

British Airways- Tableau Project

A Tableau dashboard project analyzing British Airways customer reviews from 2016 to 2023. The dashboard highlights trends in customer satisfaction across metrics like cabin service, entertainment, food, and seat comfort. It features time series analysis, aircraft-wise performance, country-level insights, and sentiment analysis using review text. This project showcases my skills in data visualization, storytelling, and interactive dashboard design.

Customer Segmentation – Clustering Project

This project segments mall customers into distinct groups using KMeans clustering based on age, income, and spending behavior. The goal is to help marketers identify high-value customers and design tailored targeting strategies. The analysis includes exploratory data analysis, optimal cluster selection using the elbow method, and business-driven recommendations based on cluster characteristics.
